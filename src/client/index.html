Welcome to the Zip to MSA API

Please send a GET request with a query parameter "zip" to "/api".

API response will include the following:
  • zip: Your input ZIP Code
  • cbsa: Corresponding CBSA (Core Based Statistical Area) Code
  • msaName: Corresponding MSA (Metropolitan Statistical Area) Name
  • population2014: MSA Population in 2014
  • population2015: MSA Population in 2015
